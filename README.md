# Portfólio Pessoal

Este projeto foi desenvolvido como atividade da disciplina de Análise e Desenvolvimento de Sistemas. O objetivo é criar uma landing page responsiva para apresentar informações pessoais, habilidades e projetos desenvolvidos.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Apresentação pessoal
- Lista de habilidades
- Exibição de projetos
- Informações para contato
- Layout responsivo

## Como executar

1. Clone este repositório:
   git clone https://github.com/seu-usuario/seu-repositorio.git

2. Abra o arquivo `index.html` em um navegador.
