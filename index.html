Projetos

Projeto MagickIA

Esse projeto foi criado a partir de um evento realizado pelo Dev em dobro, criamos o projeto do zero assistindo as lives.
Loja virtual fictícia de cartas colecionáveis desenvolvida para praticar HTML, CSS e JavaScript. O projeto conta com catálogo de produtos, filtros por categoria e preço, além de uma interface responsiva e intuitiva.

Projeto Esports NLW

Projeto criado a partir de outro evento, o NLW da Rocketseat, onde desenvolvemos um assistente de meta para criar buils e dicas de jogos.
Desenvolvimento de uma plataforma para criação de builds e consulta de dicas para jogos como League of Legends, Counter-Strike e outros. O projeto integra a API do Gemini, permitindo que o usuário faça perguntas em um campo de texto e receba recomendações personalizadas geradas por inteligência artificial.

Projeto Criador de animações com IA

O projeto foi desenvolvido através de um evento ao vivo do Dev Club, onde criamos um site que gera animações a partir de textos digitados pelo usuário.
Aplicação web para geração de animações com inteligência artificial. O usuário descreve a animação desejada em um campo de texto, e a IA gera tanto a animação em tempo real quanto o código CSS correspondente, permitindo visualizar o resultado e aprender como ele foi desenvolvido.
